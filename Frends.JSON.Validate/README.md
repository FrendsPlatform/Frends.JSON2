# Installing

You can install the Task via Frends UI Task View.

## Building


Rebuild the project

`dotnet build`

Run tests

`Tests require a valid Newtonsoft Json.NET Schema license.`
`dotnet test`


Create a NuGet package

`dotnet pack --configuration Release`
